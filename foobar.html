For this project I received a JSON file with a lot of data, from which I had to choose which parts to use and build a dashboard web app, which shows the current state of the fictitious ‘Foobar’. I created a dashboard meant to help the bar manager supervise the resources and work flow. When designing it I used the approach for creating infographics, where even if one removes all the text, it still should be clear what is the theme of the dashboard is about.
